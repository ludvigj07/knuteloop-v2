## Gotchas / things to remember

- **Windows + pnpm EPERM:** `expo install` / `pnpm add` can fail mid-extract with `EPERM: rename` (transient, Defender/lock). A plain retry works. It also **rolls back the package.json change** on failure — verify the dep actually landed (`node -e "require.resolve(...)"`).
- **jest + pnpm:** default `transformIgnorePatterns` misses pnpm's `.pnpm/` nesting → RN Flow syntax errors. Fixed in `apps/mobile/jest.config.js`.
- **api tests need a real Postgres at localhost:5432** (`postgres:postgres`), NOT testcontainers — they create `knuteloop_test` + roles via migrations. CI service provides it; locally you need Docker/Postgres up.
- **Hook bug fixed:** branch names with "-f" no longer false-trigger the force-push block (was a silent "No stderr output" block).
- **Primitive `Pressable` requires `accessibilityLabel`** (good — forces a11y). Caller's `accessibilityState` overrides the primitive's `disabled` one (fine for current callers).
- **Migration 0007 is non-CONCURRENT** — convert before it hits a populated prod table.
