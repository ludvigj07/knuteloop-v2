## Gotchas / things to remember

- **Android hard shadow:** RN `elevation` can't do a directional hard offset shadow. `StickerCard` renders a solid-ink backing `View` offset behind the surface ŌĆö that's the cross-platform trick. Don't "simplify" it to elevation.
- **Custom fonts = weight baked into family name.** The `Text` primitive sets `fontFamily` (e.g. `Inter_700Bold`) and deliberately omits `fontWeight` (pairing both = Android faux-bold). Font keys in `useFonts` must match `fontFamily` map in `theme.ts`.
- **Two token layers now:** legacy `colors`/`radius`/`shadows` (pre-sticker screens) + the `sticker` namespace (admin). Admin screens use cream `sticker.color.paper` bg; consumer screens unchanged.
- **`@gorhom` sheets** need `BottomSheetModalProvider` (added in `_layout`) ŌĆö already wired.
- **`lib/knute-ui.ts` `folderGlyph`** maps library folder names (Generelle/Dobbel/Rampestrek/Alkohol/Sex) to glyphs; Rampestrek ŌåÆ `fordervett` glyph. Sensitive folders = Alkohol/Sex (amber tint + detail-sheet-first on quick add).
- The region filter pills are seed-specific (Alle/Nasjonalt/Stavanger) ŌĆö hardcoded in `FilterBar.tsx`; revisit if more regions appear.
