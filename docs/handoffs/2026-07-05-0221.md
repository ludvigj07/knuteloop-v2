## Gotchas / things to remember

- **Stacked PR + squash = GitHub lukker barnet.** PR 3 er bevisst uåpnet. Rebase --onto main etter #85-merge.
- **FK-sjekker bypasser RLS** — derfor verifiserer PATCH /me/class og alt lignende tenant-tilhørighet eksplisitt før skriving. Mønster å gjenbruke.
- Partial-unique-indeksen tillater kun ÉN aktiv (pending/approved) innsending per (user, knute) — test-seeds må respektere det (bet meg i users.test.ts).
- Rang i `/api/users/:id` beregnes med count(*) og MÅ matche leaderboard-sorteringen (points desc, russenavn asc) — test låser det.
- `test/profil-integrasjon` er throwaway — slett den når PR-ene er inne.
- Dev-serverne (API + Metro) kjørte fortsatt i bakgrunnen ved sesjonsslutt; node-orphans kan holde port 3000/8081 — `netstat -ano | grep :3000` → `taskkill //F //PID <pid>`.
- Memory-notatene (global-feed-vision, project-timeline) er oppdatert med 2027-retarget + profilkort-konseptet — les dem før planlegging.
