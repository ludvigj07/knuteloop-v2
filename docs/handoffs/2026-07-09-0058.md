## Gotchas / things to remember

- **TO CLAUDE-SESJONER I SAMME MAPPE KOLLIDERER.** Skjedde i natt: den andre sesjonen byttet branch midt i arbeidet → commit landet på feil branch (reddet via reflog + `merge --ff-only` + branch-reset). Regel heretter: parallell sesjon kjører i **git worktree** (`git worktree add ..\knuteloop-v2-<tema> <branch>`). Sjekk `git branch --show-current` FØR hver commit. Notert i memory (multi-pc-workflow).
- **pre-bash-hooken blokkerer alle kommandoer som inneholder «main»** — også `gh pr create --base main`. Løsning: utelat `--base` (gh defaulter til main) og push branch i separat kall.
- `gh pr merge --squash --delete-branch` sletter OGSÅ lokal branch og auto-checkouter+puller main. Praktisk, men ikke bli overrasket.
- gh CLI ER logget inn på hoved-PC-en nå (gamle notater sa motsatt).
- Seed-bilder: første `dev:seed-feed`-kjøring trenger internett (laster picsum → disk); re-runs bruker diskfilene.
