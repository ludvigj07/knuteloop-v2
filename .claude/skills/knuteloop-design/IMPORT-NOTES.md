## Two files handled specially

- **`assets/app-icon.png` — NOT imported.** The MCP `get_file` read caps at 256 KiB
  and this PNG is larger, so it came back truncated (corrupt, missing its IEND chunk).
  Rather than commit a broken binary it was dropped. `guidelines/brand-icon.card.html`
  and `readme.md` reference it, so that one preview shows a broken image until the file
  is added. **To restore it:** download `assets/app-icon.png` from the design project on
  `claude.ai/design` and drop it in `assets/`.
- **`_ds_manifest.json` — skipped.** This is claude.ai Design-pane index metadata,
  regenerated by the design tooling; it isn't needed for this skill as a local reference
  and isn't referenced by any preview (only `_ds_bundle.js` is). Left out to keep the
  import clean.
