A russ's avatar — shows a photo, or falls back to initials on a deterministic gradient (same name → same colors). Optional ring colored by russType.

```jsx
<Avatar name="Sofie Sprint" size={56} ring="red" />
<Avatar name="Emil Baka" src={photoUrl} ring="blue" />
```

`ring`: `red` (rødruss), `blue` (blåruss), `accent`, `none`. Use `square` for a rounded-square variant. Sizes set via `size` (px diameter).
