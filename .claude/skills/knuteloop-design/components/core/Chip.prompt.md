Small pill label for categories, difficulty, points, and submission status.

```jsx
<Chip tone="primary" icon={<KnoteIcon name="alkohol" size={14} />}>Alkohol</Chip>
<Chip tone="accent">25 poeng</Chip>
<Chip tone="success">Godkjent</Chip>
<Chip tone="warning">Venter</Chip>
<Chip tone="danger">Avvist</Chip>
```

Status convention: `success`=Godkjent, `warning`=Venter, `danger`=Avvist. Sizes `sm | base | lg`; `outline` for a lighter treatment.
