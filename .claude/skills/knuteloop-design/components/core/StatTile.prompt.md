Compact stat readout for points, streak, rank, and completed-knute counts. Big mono numeral, uppercase label.

```jsx
<StatTile value="285" label="Poeng" tone="primary" />
<StatTile value="7" label="Dagers streak" icon={<img src="assets/icons/streak-flame.svg" width="20" />} />
<StatTile value="#3" label="Toppliste" align="center" />
```

`tone`: `card | primary | accent | soft`. Lay several in a flex/grid row for a stats strip.
